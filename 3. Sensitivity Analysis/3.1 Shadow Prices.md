# 3.1 Shadow Prices
<div align='justify'>
  Shadow price of a constraint, also known as <i>dual variables, marginal values, or pi values,</i> are the <b>change in value of objective function per unit increase in the RHS value of that constraint,</b> with all <b>other problem data remaining unchanged.</b><br><br>
  Shadow price is only valid <b>within the allowable increase/decrease range</b> of that constraint's RHS. If the change exceeds this range, the current optimal basis may change, and the <b>shadow price will no longer apply.</b><br><br>
