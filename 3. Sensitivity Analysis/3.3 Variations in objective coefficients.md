# 3.3 Variations in Objective Coefficients
<div align='justify'>
  Previously, we have looked at shadown prices and reduced cost, which are due to <b>changes in the RHS of constraints and nonnegativity constraints.</b> Now let us look at how much can <b>objective function coefficients</b> vary before changing the values of <b>decision variables</b> in the optimal solution.<br><br>
  Changes should be made <b>one at a time,</b> holding all <b>other coefficients and RHS values constant.</b><br><br>
  Let us consider the same example as before from <b>3.1 Shadow Prices</b> and <b>3.2 Reduced cost</b>:<br><br>
  <table align='center'>
    <tr>
      <td>
        Maximize z = 5x<sub>1</sub> + 4.5x<sub>2</sub> + 6x<sub>3</sub> (hundreds of dollars)
        <div align='left'>
          subject to:
        </div>
        6x<sub>1</sub> + 5x<sub>2</sub> + 8x<sub>3</sub> ≤ 60 (production capacity; hours)<br>
        10x<sub>1</sub> + 20x<sub>2</sub> + 10x<sub>3</sub> ≤ 150 (warehouse capacity; hundreds if sq. ft.)<br>
        x<sub>1</sub> ≤ 8 (demand for glass x<sub>1</sub>)
        x<sub>1</sub> ≥ 0, x<sub>2</sub> ≥ 0, x<sub>3</sub> ≥ 0<br>
      </td>
    </tr>
  </table>
  We will look at two cases, changing the coefficient of a <b>nonbasic and basic variable.</b>
</div>

## 3.3.1 Nonbasic variable
<div align='justify'>
Now if we change the coefficient of x<sub>3</sub> in the original objective function by a small amount, denoted by the equation below:<br><br>
  <table align='center'>
    <tr align='center'>
      <td>
        $$5 x_1 + 4.5 x_2 + (6+\Delta c_3) x_3 - z = 0$$
      </td>
    </tr>
    <tr align='center'>
      <td>
        where $$6+\Delta c_3$$ is denoted by $$c_{3}^{new}$$
      </td>
    </tr>
  </table>
  The new final tableau is as below, where everything <b>remains unchanged except the addition of the small change:</b><br><br>
  <table align='center'>
    <tr align='center'>
      <th>Basic variables</th>
      <th>Current values</th>
      <th>x<sub>1</sub></th>
      <th>x<sub>2</sub></th>
      <th>x<sub>3</sub></th>
      <th>x<sub>4</sub></th>
      <th>x<sub>5</sub></th>
      <th>x<sub>6</sub></th>
    </tr>
    <tr align='center'>
      <td>x<sub>2</sub></td>
      <td>$$4 \frac{2}{7}$$</td>
      <td></td>
      <td>1</td>
      <td>$$- \frac{2}{7}$$</td>
      <td>$$- \frac{1}{7}$$</td>
      <td>$$\frac{3}{35}$$</td>
      <td></td>
    </tr>
    <tr align='center'>
      <td>x<sub>6</sub></td>
      <td>$$1 \frac{4}{7}$$</td>
      <td></td>
      <td></td>
      <td>$$- \frac{11}{7}$$</td>
      <td>$$- \frac{2}{7}$$</td>
      <td>$$\frac{1}{14}$$</td>
      <td>1</td>
    </tr>
    <tr align='center'>
      <td>x<sub>1</sub></td>
      <td>$$6 \frac{3}{7}$$</td>
      <td>1</td>
      <td></td>
      <td>$$\frac{11}{7}$$</td>
      <td>$$\frac{2}{7}$$</td>
      <td>$$- \frac{1}{14}$$</td>
      <td></td>
    </tr>
    <tr align='center'>
      <td>(-z)</td>
      <td>$$-51 \frac{3}{7}$$</td>
      <td></td>
      <td></td>
      <td>$$- \frac{4}{7} + \Delta c_3$$</td>
      <td>$$- \frac{11}{14}$$</td>
      <td>$$- \frac{1}{35}$$</td>
      <td></td>
    </tr>
  </table>
  Since the <b>coefficient of x<sub>3</sub> has changed,</b> it became a <b>candidate to enter the basis.</b> The <b>optimal solution remains unchanged</b> as long as:<br><br>
  <table align='center'>
    <tr align='center'>
      <td>
        $$- \frac{4}{7} + \Delta c_3 \leq 0$$ or $$\Delta c_3 \leq \frac{4}{7}$$
      </td>
    </tr>
  </table>
  Equivalently, it means that the <b>range</b> of the <b>original objective function coefficient</b> of x<sub>3</sub> must satisfy:<br><br>
  <table align='center'>
    <tr align='center'>
      <td>
        $$- \infty \lt c_{3}^{new} \leq 6 \frac{4}{7}$$
      </td>
    </tr>
  </table>
</div>

## 3.3.2 Basic variable
<div align='justify'>
  Now consider changing the objective function coefficient of a basic variable. In this example, we will take x<sub>1</sub>.<br><br>
  When we alter the coefficient value of a basic variable, if it is <b>reduced sufficiently,</b> we may end up <b>not using it</b> (leaving the basis), and if it is <b>increased sufficiently,</b> we may end up <b>only having that basic variable.</b><br><br>
  Now let us alter the coefficient of x<sub>1</sub> in the original objective function shown below:<br><br>
  <table align='center'>
    <tr align='center'>
      <td>
        $$(5 + \Delta c_1 ) x_1 + 4.5 x_2 + 6 x_3 - z = 0$$
      </td>
    </tr>
    <tr align='center'>
      <td>
        where $$5+\Delta c_1$$ is denoted by $$c_{1}^{new}$$
      </td>
    </tr>
  </table>
  By applying the same logic as the nonbasic variable, we will get the following final tableau:<br><br>
  <table align='center'>
    <tr align='center'>
      <th>Basic variables</th>
      <th>Current values</th>
      <th>x<sub>1</sub></th>
      <th>x<sub>2</sub></th>
      <th>x<sub>3</sub></th>
      <th>x<sub>4</sub></th>
      <th>x<sub>5</sub></th>
      <th>x<sub>6</sub></th>
    </tr>
    <tr align='center'>
      <td>x<sub>2</sub></td>
      <td>$$4 \frac{2}{7}$$</td>
      <td></td>
      <td>1</td>
      <td>$$- \frac{2}{7}$$</td>
      <td>$$- \frac{1}{7}$$</td>
      <td>$$\frac{3}{35}$$</td>
      <td></td>
    </tr>
    <tr align='center'>
      <td>x<sub>6</sub></td>
      <td>$$1 \frac{4}{7}$$</td>
      <td></td>
      <td></td>
      <td>$$- \frac{11}{7}$$</td>
      <td>$$- \frac{2}{7}$$</td>
      <td>$$\frac{1}{14}$$</td>
      <td>1</td>
    </tr>
    <tr align='center'>
      <td>x<sub>1</sub></td>
      <td>$$6 \frac{3}{7}$$</td>
      <td>1</td>
      <td></td>
      <td>$$\frac{11}{7}$$</td>
      <td>$$\frac{2}{7}$$</td>
      <td>$$- \frac{1}{14}$$</td>
      <td></td>
    </tr>
    <tr align='center'>
      <td>(-z)</td>
      <td>$$-51 \frac{3}{7}$$</td>
      <td>$$\Delta c_1$$</td>
      <td></td>
      <td>$$- \frac{4}{7}$$</td>
      <td>$$- \frac{11}{14}$$</td>
      <td>$$- \frac{1}{35}$$</td>
      <td></td>
    </tr>
  </table>
  However, a basic variable can't have a nonzero coefficient in the final objective function, so we have to subtract the change times row 3 from row 4, giving the updated final tableau below:<br><br>
  <table align='center'>
    <tr align='center'>
      <th>Basic variables</th>
      <th>Current values</th>
      <th>x<sub>1</sub></th>
      <th>x<sub>2</sub></th>
      <th>x<sub>3</sub></th>
      <th>x<sub>4</sub></th>
      <th>x<sub>5</sub></th>
      <th>x<sub>6</sub></th>
    </tr>
    <tr align='center'>
      <td>x<sub>2</sub></td>
      <td>$$4 \frac{2}{7}$$</td>
      <td></td>
      <td>1</td>
      <td>$$- \frac{2}{7}$$</td>
      <td>$$- \frac{1}{7}$$</td>
      <td>$$\frac{3}{35}$$</td>
      <td></td>
    </tr>
    <tr align='center'>
      <td>x<sub>6</sub></td>
      <td>$$1 \frac{4}{7}$$</td>
      <td></td>
      <td></td>
      <td>$$- \frac{11}{7}$$</td>
      <td>$$- \frac{2}{7}$$</td>
      <td>$$\frac{1}{14}$$</td>
      <td>1</td>
    </tr>
    <tr align='center'>
      <td>x<sub>1</sub></td>
      <td>$$6 \frac{3}{7}$$</td>
      <td>1</td>
      <td></td>
      <td>$$\frac{11}{7}$$</td>
      <td>$$\frac{2}{7}$$</td>
      <td>$$- \frac{1}{14}$$</td>
      <td></td>
    </tr>
    <tr align='center'>
      <td>(-z)</td>
      <td>$$-51 \frac{3}{7} -6 \frac{3}{7} \Delta c_1$$</td>
      <td></td>
      <td></td>
      <td>$$- \frac{4}{7} - \frac{11}{7} \Delta c_1$$</td>
      <td>$$- \frac{11}{14} - \frac{2}{7} \Delta c_1$$</td>
      <td>$$- \frac{1}{35} + \frac{1}{14} \Delta c_1$$</td>
      <td></td>
    </tr>
  </table>
  In order to <b>preserve the optimality</b>, the objective function need to have <b>negative coefficients,</b> which means we must have:<br><br>
  <table align='center'>
    <tr align='center'>
      <td>
        $$- \frac{4}{7} - \frac{11}{7} \Delta c_1 \leq 0$$, (that is, $$\Delta c_1 \geq - \frac{4}{11}$$)
      </td>
    </tr>
    <tr align='center'>
      <td>
        $$- \frac{11}{14} - \frac{2}{7} \Delta c_1 \leq 0$$, (that is, $$\Delta c_1 \geq - \frac{11}{4}$$)
      </td>
    </tr>
    <tr align='center'>
      <td>
        $$- \frac{1}{35} + \frac{1}{14} \Delta c_1 \leq 0$$, (that is, $$\Delta c_1 \leq \frac{2}{5}$$)
      </td>
    </tr>
  </table>
  Thus, taking the most limiting inequalities gives:<br><br>
  <table align='center'>
    <tr align='center'>
      <td>
        $$- \frac{4}{11} \leq \Delta c_1 \leq \frac{2}{5}$$
      </td>
    </tr>
  </table>
  Equivalently, it means that the <b>objective function coefficient of x<sub>1</sub> in the initial tableau is:</b><br><br>
  <table align='center'>
    <tr align='center'>
      <td>
        $$4 \frac{7}{11} \leq c_{1}^{new} \leq 5 \frac{2}{5}$$
      </td>
    </tr>
  </table>
</div>
