# 3.0 Introduction
<div align='justify'>
  Sensitivity analysis in linear programming examines how the optimal solution changes when the input data is varied. This analysis helps understand the robustness of solution and provides insights into potential improvements or adjustments in the process being modeled, without needing to solve the problem from scratch.<br><br>
  It answer questions like:<br>
  <ul>
    <li>What happends if the <b>objective function coefficients</b> change?</li>
    <li>What happens if the <b>righthand side (RHS)</b> of constraints changes?</li>
    <li>How much can a parameter change before optimal basis changes?</li>
  </ul>
  Why is it useful?<br>
  <ul>
    <li><b>Decision making:</b> It helps managers understand the robustness of the solution.</li>
    <li><b>Cost control:</b> It shows how sensitive profit/cost is to parameter changes.</li>
    <li><b>Scenario planning:</b> Allows "what-if" analysis without re-solving the lienar program.</li>
  </ul>
  We will first explore <b>shadow prices</b> in the next section.
</div>
